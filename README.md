# Loket API

Loket API

## Getting Started

### Prerequisites

Things you need to have and installed

```
MySQL   v5.7
Node.js v12.4.0 or latest
NPM	v6.9.0 or latest
```

### Installing

Setup database

```
import script loket.sql
```

Installing dependencies

```
npm i
```

## Running the server

How to run the server

```
npm start
```

## Running the tests

How to run the automated tests

```
npm test
```
